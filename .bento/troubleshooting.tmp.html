This chapter describes the many error codes available in RA, along with their descriptions, severity, and corrective measures.

Error codes can be categorized as follow:

Machine errors: Errors that are issued based on the feedback from a machine controller.

Software errors: Errors that are issued by the HMI software used to operate the machine.

Machine Errors

Oil temperature low

Error Severity Description Corrective measures

Hydraulic oil temperature low

Warning

When the hydraulic oil temperature is low (during cold start condition), the machine shows an error. Homing of ram Y1 ans Y2 moves to the origin position.

Select the Bend 180 degree program to run the machine in Endurance Mode, until the warning disappears.

Software Errors

Flange too narrow error

Error Severity Description Corrective measures

Flange too narrow

Warning

Flange is not seated properly in the punch

Make sure proper seating of the flange over the die shoulders.
Sheet may tilt during bending and cause injury to operator.

Hemming error

Error Severity Description Corrective measures

IO Hem punch die collision error

Warning

RA version beyond 145 will show this error while using IO Hem tooling

Ram and crown corrections are auto corrected from RA 146. No actions required.
Using RA version less than 145 with IO hem punch will cause punch and die to crash, resulting in collision error.

Higher punch angle error

Error Severity Description Corrective measures

Punch angle greater than die angle

Warning

When punch angle is greater than die angle, warning message is shown for the bend

Select proper tooling for the bend.
Punch and die can lock into each other.
