Two speed pressing

Enable two speed pressing

Range

ON/OFF

Default

ON

Units

NA

Access

Administrator

Function

This parameter enables/disables the two-speed pressing option.

Description

When two-speed pressing option is enabled, the machine operator has the possibility to program an independent approach speed, from mute to the pinching point, and pressing speed.

This parameter is not validated.

This parameter defines the maximum travel speed of the Y-axis between the mute point and the pinching point.

Description

This is the maximum speed the machine operator can program for the movement from the mute point to the pinching point.

This parameter is not validated.
