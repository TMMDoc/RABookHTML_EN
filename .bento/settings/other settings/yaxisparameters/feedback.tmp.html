Function

Description

The resolution of linear scales used in Left (Y1) and Right (Y2) side of RAM is defined here and it supports 1 micron or 5 microns resolution. Please make sure that both the sides use the same resolution of linear scale.

Reference left (Y1)

Range

0-9999.99

Default

25

Units

mm

Access

Administrator

Function

This parameter defines the reference offset position of Y1-axis left side linear scale and parallelism adjustment.

Description

With respect to this value, the Y1-axis position is displayed on the screen after passing the reference mark.

This value must be greater than the mechanical travel distance on the scale from reference mark position to the scale end. This travel distance must also be greater than the mechanical travel of the cylinder from reference point position to the upper dead point mechanical in the cylinder. Make sure that when the cylinder is on the mechanical upper dead stop the Y-axis value readout is between 5 and 10 mm.

General

Y-ref search speed

Range

10-250

Default

125

Units

mm/s

Access

Administrator

Function

This parameter defines the speed with which the beam moves in upward direction during reference search cycle.

Description

When the reference search cycle of the Y-axis is started, the beam is moved in the upward direction with the speed programmed here. The sequence of the complete reference cycle depends on the position of the beam:

  • If the beam was positioned below the reference point, the Y-axis will reach the reference point and set the position.

  • If the beam was positioned above the reference point, the beam will also move upwards with the programmed speed. When the beam reaches the mechanical upper dead point, the control will detect after a short period that there is no movement and will stop the reference command. Now, when the foot pedal close is pressed, the beam start move downwards with closing speed (specified in pressing parameter).

  • During downward movement, when the beam crosses the reference mark, a stop command is issued, and the controller goes to stop mode. At this position, it is made sure that the beam is below the reference point. Now, when a start command is given the beam moves in the upward direction, crosses the reference mark and set the position of Y-axis to complete the reference cycle.

This method of reference cycle make sure that the Y-axis position is always set during the upward movement(while referencing) of the beam.

Parallelity switch

Range

No Switch / Two Switch

Default

No Switch

Units

NA

Access

Administrator

Function

Two switches mounted across Y1 & Y2 is used for Parallelism indication of the RAM used before Y-axis references are passed.

Description

If the parameter is selected as 'Two Switch', then the switch inputs are connected to digital inputs of SmartCore. With this inputs the control has an indication of the parallelism of the RAM. When this inputs are active (switch closed) the RAM parallelism is OK.

If left side switch is open, then the RAM is down in left side and up in right side or vice versa. Depends on the switch status, the RAM is moved in one side to make it parallel at lower speed and then pass the reference.

When the parallel switch is not mounted (No switch selected) the control will not use this input before the references are passed.
