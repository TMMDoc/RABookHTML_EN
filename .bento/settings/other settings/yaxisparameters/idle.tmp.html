Idle Settings

P-gain

Range

0.1-999.99

Default

0

Units

NA

Access

Administrator

Function

This parameter defines the proportional gain factor for the Y-axis during idle status.

Description

During the idle condition of Y-axis, the response to changes in the control signal should be very minimum. A high gain gives a quick response, if the gain is too high it will cause oscillation.

Timeout of idle pressure in run mode

Range

0-100

Default

0

Units

Sec

Access

Administrator

Function

This parameter specifies the time till which run mode pressure will be maintained. This parameter is set in order to avoid unnecessary motion of pump and save energy if the operator leaves the machine in run mode (when program is stopped in between).

Description

This parameter will be available only if pressure control type in Y-Axis Parameters → Pressure valve is set to Servo motor.
