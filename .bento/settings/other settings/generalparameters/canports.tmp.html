Baud rate

CAN 1

Range

500/800/1000

Default

1000

Units

kbps

Access

Standard

Function

This parameter defines the baud rate of CAN1 port communication.

Description

Generally, the CAN1 port is used for the communication of CAN slave I/O ports for the expansion and/or the RA Smart Axis card.

1000

Units

kbps

Standard

Function

This parameter defines the baud rate of CAN2 port communication.

Description

Generally, the CAN2 port is used for the communication of CAN drives for Backgauge axes.
