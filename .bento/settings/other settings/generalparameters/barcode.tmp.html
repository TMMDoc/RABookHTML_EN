Function

Before setting up bar code scanner, it needs to be recognized by RA software. Steps to register a bar code scanner in RA :

  1. Click on the 'Select device' button in the Bar code parameters page.

  2. Now device selection tab appears, now scan any bar code using the scanner.

  3. The input text can be checked, and device ID is assigned automatically.

  4. Now close the machine parameter page and save changes made.
