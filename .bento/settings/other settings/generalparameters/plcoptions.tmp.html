Options

Safety

Range

ON/OFF

Default

OFF

Units

Access

Service

Function

This parameter activates/deactivates the laser safe functionality in the PLC.

Description

When this parameter is ON, the laser safe functions are activated in the PLC, such as Enable functions, Side door functions etc.., If this parameter is OFF, then the 'Side Door' options will be available for selection.

Side doors

Range

1 input / 2 inputs

Default

1 input

Units

Access

Service

Function

This parameter defines the number of digital inputs used for the configuration of side doors.

Description

This parameter is available only if the safety option is OFF. If the left and right-side doors are configured in 2 digital inputs then '2 inputs' selected, else if the left and right doors are combined in hardware circuit and mapped to 1 digital input, then '1 input' is selected.

Cold

Range

ON/OFF

Default

OFF

Units

Access

Service

Function

This parameter activates/deactivates the cold condition functionality in PLC.

Description

When this parameter is ON, the soft key 'Cold On/Off' is available in the options soft key page. If this parameter is OFF, the soft key will disappear and the functionality in the PLC is also deactivated.

When the Cold soft key is switched ON, the cold condition parameter of the Y-axis is activated. Also, if the temperature is below the normal working condition then this function is activated automatically and keeps the soft key always high.
