Safety method

Range

Normal / No safe guard

Default

Normal

Units

Access

Service

Function

This parameter is used to specify whether bend guard is used or not.

Description

By default, normal is chosen which means bend guard is used. If “No safe guard” is chosen, then machine is considered without bend guard.

Serial port

Range

COM1/COM2

Default

COM1

Units

Access

Service

Function

This parameter defines the COM number of serial port to which the safety external device is connected. Such a device can be a PLC, safety PLC or an angle measurement system. Right now, only one safety system is supported as follows:

  • LaserSafe PCSS

If a device is connected, the control and the device can exchange status information or press commands across the serial connection.
