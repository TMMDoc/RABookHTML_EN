Status display

In Service row, the status display is showed in numeric screen of run page at the Left side bottom as indicated in the picture. The status display 3.8.0 can be divided into three segments as follows:

  1. Controller status

  2. Machine status

  3. Program run status

Controller status: The number and the corresponding meaning of status are as follows:

0

Controller is in Fatal error status and not recoverable until power restart.

1

Controller is in Fatal error status and recoverable.

2

Controller is in Normal error status and recoverable.

3

Controller is in Ready status for operation.

Machine Status: The number and the corresponding meaning of status are as follows:

0 - 7

The state from 0 to 8 displays the booting mode of the machine with defined parameters.
If the booting mode is stopped in between 1 & 7, the corresponding message will be displayed.

8

Machine Booting process is completed.
It may be noted that, after the machine status reaches the state 8, the Stop LED (red) lights ON

9

Backgauge axes are enabled.

10

RAM and Backgauge axes are ready for operation.

Program run Status: The number and the corresponding meaning of status are as follows:

0

Waiting for Start command (or) Axes are not referenced.

1 - 3

Internal program functions.

4

Wait for foot pedal close trigger.
It may be noted that if the auto is selected for Backgauge move, then this step will be ignored

5 - 7

Internal parameter functions.

8

Active motion of all axes until they reach their respective target positions.

9

Dwell time.

10

Internal function.

11

Decompression start.

12

Decompression end.

Force & Crown values: In service row, additionally the Force and Crown, programmed & actual values are displayed in numeric screen of run page at the bottom of axis positions as marked in the picture.

Force values are displayed in units of tons and Crown values are displayed in units of micrometre (µm).

Units

Function

This parameter defines the stiffness of the frame used to compute the Y-axis depth calculation.

Description

The stiffness of frame is used to compute the deflection of the side frame according to the programmed force. Thereby the compensation distance due to deflection of side frame is adjusted in Y-axis depth calculation.

This parameter is programmed as a percentage of the value programmed for the machine parameter 'Relaxation Stiffness'.

This parameter enables the handwheel function in the machine.

Description

Using handwheel, RAM (Y-Axis) and all the backgauge axes are moved in a very precise manner. It is mainly used during manual clamping of tools, where the pressure is set to minimum, the punch and die brought in contact with each other and then clamped to make sure the centre between the tools are aligned.

Sheet Support

Range

ON/OFF

Default

OFF

Units

NA

Access

Administrator

Function

This parameter enables the sheet follower function in the machine. Only if this parameter is ON the S1 and S2 axis settings will be displayed.

Description

Sheet support is used to support the part and reducing human effort and increasing bend accuracy. Kindly note that to use sheet support function, sheet follower license needs to be enabled in license feature list.

Sheet support implementation is as follows:

  • S1 and S2 will be move to 0 when back gauge is positioned irrespective of whether bend required sheet support to move. This is to ensure sheet support is in flat condition when bending a part.

  • If the sheet support is enabled corresponding S1 or S2 or both will move while bending.

  • After reaching TDC, S1 or S2 or both will return to flat position if it is enabled.

Axis parameter for S1 and S2 are similar to other axis except for minor changes like the minimum and maximum unit is degrees.
