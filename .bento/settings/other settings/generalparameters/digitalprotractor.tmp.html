Function

Before setting up angle measuring device, it needs to be recognized by RA software. Steps to register an angle measuring device in RA :

  1. Click on the 'Select device' button in the Digital Protractor parameters page.

  2. Now device selection tab appears, now press the data button in the device.

  3. The input angle can be checked, and device ID is assigned automatically.

  4. Now close the machine parameter page and save changes made.

Angle tolerance

Range

Default

5

Units

Degree

Access

Administrator

Function

Angle tolerance is used when Auto correct angle is enabled. If the measure angle deviate from actual angle by more than angle tolerance, auto correct angle is disabled, and user need to apply the correction manually. Angle Input Sequence in RA :

  1. For smaller length bend length

    • Only central angle correction is available and angle from protractor is transferred to the central angle.

  2. If Auto correct angle is enabled, for each data transfer angle correction will be applied.

    • Otherwise, central angle will be overwritten.

  3. For longer bend length

    • Data transfer in run page will open angle correction page with three angles correction mode activated and all the angles will be populated with transferred angle.

    • Next data transfer will be entered in right end angle with central angle value equal to average of end angles.

    • Next data transfer will be entered in central angle.

  4. Mode is not changed automatically from central angle to end angle or from end angle to three.

For longer length program (length greater than Default → Bend cycle → Two / Three angle correction bend length limit), user should measure left angle first, then right angle and then central angle. After three measurement, angle correction will be applied automatically if the option is enabled. On each successful data transmission, mode will change from central angle, end angle and three angle correction.
