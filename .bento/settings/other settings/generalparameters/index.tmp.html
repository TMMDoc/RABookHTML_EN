This page explains the general parameter settings of the machine.

General settings:

Change access - To change the user access level.

Change password - To change the user access password.

Change Machine - To change machine from the available machine list.

Import machine - To import a machine using file selection panel.

Load NC map - To load an existing NC file using file selection panel.

Save NC map - To save NC file.

General settings tabs are available based on the user access level.
