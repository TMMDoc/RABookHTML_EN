General

Show IO in run screen

Range

Checked / Unchecked

Default

Unchecked

Units

Access

Service

Function

This parameter enables IO status display in run screen.

Description

IO status can be viewed in run screen as well as by clicking 'Show IO' button in Diagnostic page in machine parameter page. In run page, instead of displaying information about part, IO status is displayed. To see IO status in run page, following condition must be satisfied:

  1. User level should be Service

  2. Service row should be on (General → Machine → Service row)

  3. 'Show IO in ram screen' should be checked in General → Diagnostic
