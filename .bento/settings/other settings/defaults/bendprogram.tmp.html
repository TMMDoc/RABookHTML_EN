Deflection

Deflection method

Range

Linear /Calibration /Theory

Default

Theory

Units

Access

Administrator

Function

This parameter is used to select the method used for deflection computation.

  1. If calibration is selected, deflection is derived from deflection table and deflection factor set.

  2. If theory is selected, deflection value is calculated theoretically using stiffness and rigidity factor.

  3. If linear is selected, deflection is computed using stiffness of frame and relaxation stiffness values in machine parameter (General parameters → Machine tab).

Description

Angle correction factor

Range

0.5 to 1.5

Default

1

Units

Access

Administrator

Function

While entering angle correction, the difference between desired angle and measured angle is multiplied by this parameter value internally. For example, if angle correction factor is 1.1, desired angle is 90 deg and measured angle is 95 deg, the correction is applied for angle difference of 5.5 deg (5x1.1=5.5).

Description

Function

Selecting Basic does two angle correction and then adjusts crowning. Advanced applies angle correction using parabolic approach. It is recommended to use Basic method for machines with hydraulic crowning.
