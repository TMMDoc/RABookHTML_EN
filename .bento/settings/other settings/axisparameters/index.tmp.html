This chapter describes the parameters of the backgauge axis, handwheel and regrip axis (if used).

Parameters with the indication behind the name are always visible, but they can only be edited when you have entered with administrator or service level password.

Parameters with the indication behind the name are only accessible when service level password has been entered.

Backgauge Axis configuration

The RA smart controls are smart enough to configure the backgauge axis by themselves. With the help of machine ID, the backgauge axis configuration are derived from the machine file. In this way, the backgauge axis are perfectly positioned depending on the job, checked for collision, simulated accurately etc.,
